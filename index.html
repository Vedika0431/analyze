Welcome to the Data Processing Project

\n

\n This project demonstrates a robust data processing pipeline using Python (Pandas)\n and automates its execution and deployment via GitHub Actions.\n

\n

\n The core functionality involves reading data from an Excel file (data.xlsx),\n converting it to a CSV format (data.csv), performing some basic data transformation,\n and finally outputting the results into a JSON file (result.json).\n

\n

\n The entire process is orchestrated by a continuous integration/continuous deployment (CI/CD)\n workflow on GitHub Actions, which also publishes the generated result.json to\n GitHub Pages for easy access.\n

How it Works

\n

\n Upon every push to the main branch, the GitHub Actions workflow triggers.\n It first runs a linter (Ruff) to ensure code quality. If linting passes,\n it executes the execute.py script. This script reads data.xlsx,\n generates data.csv, processes the data, and outputs result.json.\n Finally, result.json is deployed to GitHub Pages, making it publicly available.\n
